# Labubu Images Folder

Place your Labubu images in this folder. The following images are used in the website:

## Required Images:

1. **labubu-icon.png** (or .jpg, .svg)
   - Used in the header title
   - Recommended size: 64x64px or larger
   - Should be a cute Labubu character icon

2. **labubu-smile.png** (or .jpg, .svg)
   - Used in the celebration animation when all tasks are completed
   - Recommended size: 200x200px or larger
   - Should be a happy/smiling Labubu character

3. **labubu-empty.png** (or .jpg, .svg)
   - Used when there are no tasks
   - Recommended size: 150x150px or larger
   - Should be a cute Labubu character

4. **labubu-celebration.png** (or .jpg, .svg)
   - Used in celebration effects
   - Recommended size: 100x100px or larger
   - Should be a festive Labubu character

## Image Formats Supported:
- PNG (recommended for transparency)
- JPG/JPEG
- SVG (scalable vector graphics)
- WebP

## Tips:
- Use transparent backgrounds (PNG) for better integration
- Keep file sizes reasonable for fast loading
- Ensure images match the Labubu aesthetic (cute, colorful, playful)
